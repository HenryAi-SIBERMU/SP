# Documentation Directory

Folder ini berisi dokumentasi metodologi riset, meeting notes, dan working documents.

## 📂 Struktur Folder

### `methodology/`
Dokumentasi metodologi penelitian.

**Files**:
- `gis-approach.md` - Pendekatan GIS untuk inventarisasi area
- `calculation-formulas.md` - Formula dan asumsi perhitungan
- `data-sources.md` - Sumber data dan cara akses
- `validation-approach.md` - Metode validasi hasil

### `meetings/`
Meeting notes dan stakeholder consultation.

**Format**:
```
YYYY-MM-DD_[stakeholder]_meeting_notes.md
```

**Contoh**:
- `2026-07-15_TransJakarta_meeting_notes.md`
- `2026-08-20_PLN_consultation_notes.md`

### `literature-review/`
Summary paper dan literatur yang sudah dibaca.

**Format**:
- Summary paper dalam markdown
- Annotated bibliography
- Key findings extraction

### `project-plan/`
Dokumen perencanaan project.

**Files**:
- `timeline.md` - Timeline dan milestones
- `task-breakdown.md` - Task breakdown structure
- `resource-allocation.md` - Resource dan budget planning
- `risk-register.md` - Risk identification dan mitigation

```markdown
2. ...

## Discussion Points
### Topic 1
- ...

- [Link to presentation/documents]
```
